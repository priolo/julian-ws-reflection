# julian-nodes

A TypeScript Node.js library with ESM support and Jest testing.

## Requirements

- Node.js >= 22.0.0

## Installation

```bash
npm install julian-ws-reflection
```

## Usage

```typescript
```

## Development

1. Install dependencies:
```bash
npm install
```

2. Run tests:
```bash
npm test
```

3. Build the library:
```bash
npm run build
```
